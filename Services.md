### Our Services

We currently provide two types of renting on our platform
  1.  Dapps renting ( we currently have implemented two types of dapp renting that is NFT centered )
        
        -   Rent Dapp for sale of NFT collection
        -   Rent Dapp for whitelisting of NFT collection
  2.  NFT Renting
        - One of the most important and unique selling feature for the upcoming Future of metaverse.
        - You , being an owner can upload your NFT for rent and can earn passive income by it.
        - Being an NFT lover , you can rent an NFT for some time and use it for Following use-cases
        
            - Use in games like sandbox or axie inifinity if the NFT is of `Collectibles` type.
              ( we are currently working on integration with games )
            - Use in metaverse to host an event on if the NFT is of `Real-Estate` type.
            - Build buildings with it if the NFT type is 'building'.
            - Play it in metaverse if it has the type 'music'.
              and a lot more are coming..
